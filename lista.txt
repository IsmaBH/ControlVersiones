1. git --version
2. git config --global user.name (Usar "Nombre de usuario de github" para setear uno diferente)
3. git config --global user.email (Usar "Correo de la cuenta de github" para setear uno diferente)
4. git init
5. git config --global init.defaultBranch main
6. git clone [url]
7. git status
8. git add [archivo]
9. git commit -m ""
10. git log
11. git rm --catched file
12. git pull origin master --allow-unrelated-histories (Para cuando se crean repos en diferentes fechas)
